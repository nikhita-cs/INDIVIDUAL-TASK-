##  About the Report

This repository contains my Individual Task 3 report on *Feature Extraction in Machine Learning*.

The report presents a thought experiment using an image dataset (photos of objects such as cats, dogs, cars, and trees) and explains how meaningful features improve model performance.

---

##  Key Concepts Covered

- Introduction to Feature Extraction  
- Image Dataset Description  
- Important Features in Images:
  - Colour Features  
  - Edge Features  
  - Shape Features  
  - Texture Features  
  - Spatial Features  
  - Statistical Features  
- Importance of Feature Selection  
- Impact of Feature Extraction on Model Accuracy  
- Real-World Applications  

---

##  Learning Outcomes

The report demonstrates how:

- Feature extraction reduces dimensionality  
- Meaningful features improve model accuracy  
- Feature selection prevents overfitting  
- Computational efficiency increases  
- Model interpretability is enhanced
